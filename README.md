## ğŸ“¦ How to Use

To add this datepicker to your own project:

1. Include jQuery

2. Include jQuery UI

3. Initialize the datepicker:

```javascript
$(function() {
  $("#datepicker").datepicker();
});
```

Place an input field with the matching ID:

```html
<input type="text" id="datepicker" placeholder="Select a date">
```

That's it!
